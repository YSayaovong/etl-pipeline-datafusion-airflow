# Problem Statement

You are tasked with creating a data pipeline to extract employee data from various sources, mask sensitive information within the data, and load it into BigQuery. Additionally, you are required to develop a dashboard to visualize the employee data securely.

## Requirements

### 1. Data Extraction
Extract employee data from multiple sources such as:
- Databases
- CSV files
- APIs

### 2. Data Masking
Identify and mask sensitive information within the employee data, including:
- Social Security Numbers
- Salary details
- Personal contact information

### 3. Data Loading into BigQuery
Design a secure process to load the extracted and masked employee data into Google BigQuery.

### 4. Dashboard Visualization
Develop a web-based dashboard using appropriate visualization tools such as:
- Google Data Studio (Looker Studio)
- Tableau
- Custom dashboards

The dashboard must visualize employee data securely with proper access controls.
